## 📌 Topics Covered

Each section includes problems, concepts, and sometimes optimized code solutions.

### ✅ 1. Arrays
- Basics and traversal
- Prefix sum, Kadane's Algo
- Sliding window technique
- Two-pointer approach
- Sorting and searching variations

### ✅ 2. Strings
- Character frequency
- String reversal, rotation, palindrome
- Anagram problems
- Pattern matching (KMP, Z algorithm)

### ✅ 3. Linked List
- Singly & Doubly Linked List
- Reversals, cycle detection (Floyd’s)
- Merge two sorted lists
- LRU cache (using DLL + HashMap)

### ✅ 4. Stacks & Queues
- Stack operations
- Infix/Postfix/Prefix conversions
- Next Greater Element
- Queue using stack, circular queue
- Monotonic Stack/Queue

### ✅ 5. Hashing
- HashMap & HashSet operations
- Frequency maps
- Count pairs with given sum
- Subarray sum with zero sum

### ✅ 6. Recursion & Backtracking
- Subsets and permutations
- N-Queens, Sudoku solver
- Rat in a maze
- Recursion tree analysis

### ✅ 7. Binary Search
- Lower/Upper bounds
- Search in rotated array
- Peak element
- Binary search on answer (min pages, allocation problems)

### ✅ 8. Greedy Algorithms
- Activity selection
- Fractional knapsack
- Job sequencing
- Huffman encoding

### ✅ 9. Bit Manipulation
- Set, unset, toggle bits
- Count set bits
- XOR problems (missing number)
- Power of 2 check

### ✅ 10. Sliding Window
- Max sum subarray of size k
- Minimum window substring
- Longest substring without repeating characters

### ✅ 11. Two Pointers
- Pair sum, triplet sum
- Container with most water
- Merge sorted arrays

### ✅ 12. Heaps & Priority Queue
- Heapify and custom comparators
- Top K elements
- Median in a stream
- K closest points

### ✅ 13. Trees & Binary Trees
- Inorder, preorder, postorder
- Height, diameter, balanced check
- Level order traversal
- Lowest Common Ancestor
- Binary Search Trees (BST ops)

### ✅ 14. Tries
- Insert, Search, Delete words
- Prefix search
- Word Dictionary
- Auto-complete system

### ✅ 15. Graphs
- Representations: Adj list/matrix
- DFS & BFS traversal
- Connected components
- Topological sort
- Shortest path (Dijkstra, Bellman-Ford)
- MST (Prim’s, Kruskal’s)

### ✅ 16. Dynamic Programming
- 0/1 Knapsack, Subset sum
- Longest Common Subsequence
- Matrix Chain Multiplication
- LIS, LCS, Edit Distance
- DP on Trees, Bitmasking

### ✅ 17. Sliding Window + Prefix Sum Combo
- Subarray sum equals K
- Count of subarrays with given XOR
- Max average subarray

### ✅ 18. Disjoint Set Union (DSU)
- Union by rank & path compression
- Cycle detection in graph
- Kruskal’s algorithm
- Number of islands II

### ✅ 19. Segment Tree & Fenwick Tree
- Range queries (sum/min/max)
- Point updates
- Lazy propagation

### ✅ 20. Miscellaneous
- Top interview problems
- Practice contests
- Coding assessments
